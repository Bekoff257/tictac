# SaaS Expansion Roadmap

## Phase 1
- Ranked matchmaking, shop, admin moderation, season resets.

## Phase 2
- Guilds/clans, battle pass, sponsored tournaments.

## Phase 3
- Creator economy (UGC themes), web spectator client, ad mediation optimization.

## Monetization
- Gem packs via IAP.
- Premium subscription (exclusive cosmetics, tournament perks).
- Seasonal pass and limited-time bundles.
